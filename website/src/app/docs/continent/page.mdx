# Continent codes

Below the list of continents code available. These codes can be used for the continents property of MuiTelInput.

| Code | Continent     |
| ---- | ------------- |
| AF   | Africa        |
| AS   | Asia          |
| EU   | Europe        |
| NA   | North America |
| OC   | Oceania       |
| SA   | South America |
